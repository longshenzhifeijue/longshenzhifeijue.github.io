Github

GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub。GitHub 于 2008 年 4 月 10 日正式上线，除了 Git 代码仓库托管及基本的 Web 管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。目前，其注册用户已经超过 350 万，托管版本数量也是非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。2018 年 6 月 4 日，微软宣布，通过 75 亿美元的股票交易收购代码托管平台 GitHub。

常用词

  • watch:会持续收到该项目的动态
  • fork:复制某个项目到自己到Github仓库
  • star:可以理解为点赞
  • clone:将项目下载到本地
  • follow:关注你感兴趣到作者,会收到他们的动态

in关键词限制范围

公式
关键词+in: name
关键词+in:description
关键词+in:readme
组合使用
关键词+in:name,readme

stars或fork数量关键词去查找

公式
关键词+stars+:>=数字,比如springboot stars:>=5000
关键词+forks+:>=数字,比如springcloud forks:>=5000
组合使用
查找fork在100到200之间并且stars在80到100之间到springboot项目
springboot forks:100..200 stars:80..100
