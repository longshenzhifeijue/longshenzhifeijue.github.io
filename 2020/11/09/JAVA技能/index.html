-Xms 等价于 -XX:InitialHeapSize(类似数据库表别名)(初始内存1/64)
-Xmx 等价于 -XX:MaxHeapSize(类似数据库表起别名)(初始内存1/4)

四大:引用计数,复制,标记清楚,标记整理是理论,垃圾回收器是算法的落地,只有针对具体应用的选择最合适的垃圾回收器
串行,并行,并发,G1,

G1垃圾回收器是将内存分割成不同对区域然后对其进行垃圾回收
串行和并行
查看默认垃圾收集器
java -XX:+PrintCommandLineFlags -version

G1是一款面向服务端应用对收集器,应用在多处理器和大容量内存环境中,实现高吞吐量的同时,尽可能满足垃圾收集暂停时间的要求,
另外,他
不会产生内存碎片,
可以精确控制停顿(把整个堆划分多个固定大小的区域,每次根据允许停顿的时间去收集垃圾最多的区域,)
-XX:+UseG1GC -Xmx32g -XX:MaxGCPauseMillis=100
